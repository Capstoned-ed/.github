Our Projects

RegQuest
A web-based credential request and tracking portal for managing credential requests, request processing, and workload forecasting.

Features
- Online document requests
- Request tracking
- Request queue management
- Processing calendar
- Role-based access control
- Workload forecasting using machine learning

Technologies
- React.js
- Django REST Framework
- JWT Authentication
- Scikit-learn

Mission
To develop practical and innovative systems that improve efficiency and user experience in educational institutions.
